## Keyframes 
AliasCSS provide to custom attribute  'keyframes-[name]' which can be used to define CSS keyframes declaration.

```html 
      <div  class="bgc-ecf0f1 oh w100p h100p pa t0 l0 df aic acc jcc zi-100000" 
        keyframes-loading="@100-transform-scale2 @100-border-color-gray600">
            <span class="dot" class-dot="br100p b5px-solid-555 m10px an-loading adu0d6s atf-eio ada aici "></span>
            <span class="dot adl0d2s"></span>
            <span class="dot adl0d4s  "></span>
        </div>
```
